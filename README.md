## yarn-update

Interactive update for [Yarn](https://yarnpkg.com/)

<img src="http://i.imgur.com/HmVMvYc.png" alt="yarn-update" width="386">

### Requirements
* Node >= 5.0.

### Install
```bash
$ yarn global add yarn-update
```

### Use
```bash
$ yarn-update
```
